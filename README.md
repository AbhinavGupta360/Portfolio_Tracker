# Portfolio Tracker

---

A **live portfolio tracker** built with Python and Streamlit.  

You can track your NSE stocks and analyze them with live prices, profit & loss and return metrics.

---

## Features

- Add holdings **manually** or **upload a CSV**.

- Automatic **symbol normalization** (`.NS` suffix added automatically).

- Fetch **live prices** from Yahoo Finance.

- Compute:
  > Invested Value
  > Current Value
  > Profit & Loss
  > Return %
  
- Save portfolio for future use.
